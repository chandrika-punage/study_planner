# 📚 AI Study Planner

AI Study Planner is a modern full-stack study management web application designed to help students organize their learning journey, manage daily study sessions, track goals, and monitor progress effectively.

## 🚀 Features

### 🏠 Dashboard
- Overview of study activities
- Quick access to planner sections
- Progress tracking cards
- Motivational and productivity-focused UI

### 📅 Study Planner
- Create and manage daily study schedules
- Morning, Afternoon, Evening, and Night routines
- Mark tasks as completed
- Automatic progress updates

### 🎯 Goals Management
- Add and track learning goals
- Categorized technology goals
- Progress indicators
- Weekly goal tracking

### 📊 Progress Tracking
- Visual completion indicators
- Task completion statistics
- Weekly study performance monitoring

### ⚙️ Settings
- Personalized study preferences
- User-friendly customization options

### 🎨 Modern UI
- Responsive design
- Tailwind CSS styling
- Interactive components
- Smooth animations and transitions

## ⚡ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/chandrika-punage/study_planner.git
```

### 2. Navigate to Project Directory

```bash
cd study_planner
```

### 3. Install Frontend Dependencies

```bash
cd client
npm install
```

### 4. Install Backend Dependencies

```bash
cd ../server
npm install
```

### 5. Configure Environment Variables

Create a `.env` file inside the server folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 6. Run Backend

```bash
npm start
```

### 7. Run Frontend

```bash
cd client
npm run dev
```
